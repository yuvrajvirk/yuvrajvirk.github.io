Welcome wary wanderer of cyberspace! I'm Yuvraj (pronounced "yoo-v-raj"). I'm broadly interested in building tools for developing sophisticated software by naturally communicating high-level intent to computers.

To this end, I research LLMs for code and LLM-based tools for both end-users and developers. This includes methods to more effectively and efficiently verify LLM outputs.

Research

Calibration of Large Language Models on Code Summarization
Yuvraj Virk, Premkumar Devanbu, and Toufique Ahmed
(FSE 2025) https://arxiv.org/pdf/2404.19318

LLMs are pretrained to imitate humans. Using their token probabilities, we develop a well-calibrated confidence of whether an LLM-generated code summary is similar to what a developer would've wrote.

Non-programmers Assessing AI-Generated Code: A Case Study of Business Users Analyzing Data
Yuvraj Virk, Dongyu Liu
Preprint

LLMs serve as a data analysis tool for non-technical users, but they produce misleading results. We assessed marketing and sales professionals' ability to critically evaluate LLM-generated analyses of marketing data.
